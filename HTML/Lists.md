## Unordered list:

```
<ul>
	<li>Text</li>
	<li>Text</li>
	<li>Text</li>
</ul>
```

## Ordered list:

```
<ol>
	<li>Text</li>
	<li>Text</li>
	<li>Text</li>
</ol>
```

Makes a list that looks like this:

![[ordered_list.png]]

## Nested lists:

```
<ul>
	<li>first item</li>
	<li>
		Second Item
		<ol>
			<li>second item first subitem</li>
			<li>second item second subitem</li>
			<li>second item third subitem</li>
		</ol>
	</li>
	<li>third item</li>
</ul>
```

Makes a list looks like this:
